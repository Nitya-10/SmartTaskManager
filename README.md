# Smart Task Manager

## Project Description
Smart Task Manager is a web application developed using Google Antigravity's multi-agent workflow.

## Features
- Add tasks
- Complete tasks
- Delete tasks
- Pending and completed sections
- Task counters
- LocalStorage support

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Antigravity

## Agents Used
1. Planning Agent
2. Frontend Agent
3. Backend Agent
4. Integration Agent
